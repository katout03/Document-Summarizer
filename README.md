# Intelligent Document Summarizer

An AI-powered web application designed to instantly summarize long-form English text into concise, meaningful paragraphs, helping users overcome information overload.

## 🚀 Features
* **State-of-the-Art NLP:** Utilizes the `distilbart-cnn-12-6` model from the Hugging Face Transformers library for high-accuracy text summarization.
* **User-Friendly Interface:** Built with Gradio to provide a seamless and interactive web UI.
* **Efficient Processing:** Optimized to generate summaries with dynamic length controls (`max_length` and `min_length`).

## 🛠️ Tech Stack
* **Language:** Python
* **AI/ML:** Hugging Face (Transformers), PyTorch
* **Frontend/UI:** Gradio
